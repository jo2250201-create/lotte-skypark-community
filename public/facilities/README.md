# 시설 사진

이 폴더에 시설 슬러그와 같은 이름의 이미지 파일을 넣으면 시설 카드와 상세 페이지에 자동으로 반영됩니다.
코드 수정이 필요 없습니다.

## 파일명 규칙

`{슬러그}.jpg` (또는 `.jpeg` / `.png` / `.webp`)

| 슬러그 | 시설 | 파일명 예시 |
|---|---|---|
| pool | 수영장 | `pool.jpg` |
| fitness | 헬스장 | `fitness.jpg` |
| golf | 골프연습장 (실내골프클럽) | `golf.jpg` |
| sauna | 사우나 | `sauna.jpg` |
| studyroom | 독서실 | `studyroom.jpg` |
| gym | 실내체육관 · 탁구장 | `gym.jpg` |
| kidscafe | 키즈카페 | `kidscafe.jpg` |
| kidsgym | 키즈짐 | `kidsgym.jpg` |
| programs | 커뮤니티 프로그램 | `programs.jpg` |

`cafeteria`는 이번 버전 안내에서 제외되었지만 슬러그는 남아 있습니다.

## 원칙

사진은 장식이 아니라 시설을 이해시키는 보조 자료입니다. 화질이 낮거나 시설을 대표하지 못하는 사진은
넣지 않는 것이 낫습니다 — 파일이 없으면 카드/상세 페이지는 아이콘만으로 자연스럽게 표시됩니다.
